# InsightStream Landing

Одностраничный лендинг на Docusaurus для GitHub Pages.

## Запуск локально

```bash
cd site/landing
npm install
npm run start
```

## Сборка

```bash
cd site/landing
npm run build
```

## Переменные окружения

- `LANDING_GH_OWNER` (по умолчанию `redsquad-tech`)
- `LANDING_GH_REPO` (по умолчанию `is-goose`)
- `LANDING_CONTACT_EMAIL` (по умолчанию `bavadim@gmail.com`)
- `LANDING_SITE_URL` (опционально)
- `LANDING_BASE_URL` (опционально, по умолчанию `/<repo>/`)
